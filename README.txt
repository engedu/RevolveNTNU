# Dashboard Revolve NTNU 2013/2014
This repository contains the HW and SW projects for the dashboard in Norway's first electric race car, KOG Arctos R. The project is a part of the student project Revolve NTNU(http://www.revolve.no/). Revolve NTNU is developing a new race car every year to compete in the worlds biggest engineering competition, Formula Student.

Final_code directory contains the Atmel Studio project files. The C code was running on an Atmel UC3C microcontroller.
Final_PCB_layout contains the Altium Designer project files. Both schematics and PCB layout.

The project was done in the period August 2013 to July 2014.
